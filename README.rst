Installation
~~~~~~~~~~~~

Clone the git repository:

.. code:: console

   $ git clone https://github.com/gavindsouza/inventory-frappe.git
   $ cd inventory-frappe

Install necessary dependencies

.. code:: console

   $ pip3 install -r requirements.txt


Usage
~~~~~

.. code:: console
