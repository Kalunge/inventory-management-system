Installation
~~~~~~~~~~~~

Clone the git repository:

.. code:: console

   $ git clone https://github.com/gavindsouza/inventory-frappe.git
   $ cd inventory-frappe

Install necessary dependencies

.. code:: console

   $ pip install -r requirements.txt

Go ahead and install as follows:

.. code:: console

   $ python setup.py install
